# App Store申請チェックリスト

## アプリ提出前チェック

### コード
- [ ] デバッグボタン・デバッグ関数がコメントアウトされている
- [ ] `flutter build ios --no-codesign` でビルドエラーなし
- [ ] バージョン番号・ビルド番号を更新済み

### App Store Connect
- [ ] スクリーンショット（iPhone 6.9インチ: 1320×2868px）
- [ ] アプリ説明文（日本語・英語）
- [ ] キーワード
- [ ] サポートURL（プライバシーポリシーURL）
- [ ] 年齢制限設定
- [ ] カテゴリ設定（エンターテインメント）
- [ ] 著作権情報
- [ ] データ収集の設定（プライバシー）

### IAP
- [ ] `premium_monthly` をバージョンページの「アプリ内購入とサブスクリプション」に紐づけ
- [ ] App Reviewスクリーンショット添付
- [ ] 価格設定済み

### 審査情報
- [ ] ログイン情報不要（チェック済み）
- [ ] メモ欄（カメラ使用目的など）記載

## Sandboxテスト手順

1. App Store Connect → ユーザーとアクセス → Sandboxテスター → テスター作成
2. TestFlightからアプリインストール
3. 課金ボタンタップ → Sandboxダイアログが自動表示
4. Sandboxテスターアカウントでサインイン（実際の課金なし）

## ビルド＆アップロード手順

```bash
flutter build ipa
xcrun altool --upload-app -f build/ios/ipa/*.ipa -t ios -u "minokohei@gmail.com" -p "<app-specific-password>"
```

## Apple審査リジェクト対応パターン

### 4.3(b) スパム判定
鏡機能（カメラをリアルタイム鏡として使用）が主機能であることを強調して返信。

### 2.1(b) IAP未提出
バージョンページで「アプリ内購入とサブスクリプション」にIAP商品を紐づけて再提出。

### 顔データの質問
「カメラはリアルタイムミラー表示のみに使用。顔データの収集・保存・送信は一切行わない」と返信。
